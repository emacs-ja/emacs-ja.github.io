1 Introduction💡序文

Org is a mode for keeping notes, maintaining TODO lists, and doing project planning with a fast and effective plain-text system. It is also an authoring and publishing system, and it supports working with source code for literal programming and reproducible research.
💡Orgは、メモを取ったり、TODOリストを管理したり、高速で効果的なプレーンテキストシステムでプロジェクト計画を行うためのモードです。また、 オーサリングと出 版のシステムでもあり、リテラルプログラミングや再現可能な研究のためのソースコードでの作業もサポートします。

This document is a much compressed derivative of the (org)comprehensive Org mode manual. It contains all basic features and commands, along with important hints for customization. It is intended for beginners who would shy back from a 200 pages manual because of sheer size.
💡このドキュメントは、(org)comprehensive Org mode manualをかなり圧縮して作成したものです。すべての基本的な機能とコマンド、そしてカスタマイズのための重要なヒン トが含まれています。このドキュメントは、200ページにも及 ぶマニュアルに尻込みしてしまうような初心者のためのものです。

Installation💡インストール

Important: If you are using a version of Org that is part of the Emacs distribution, please skip this section and go directly to Activation.
💡重要：Emacsの配布物の一部であるOrgのバー ジョンを使用している場合は、このセクションをスキップして、直接ア クティベーションに進んで ください。

If you have downloaded Org from the web, either as a distribution ‘.zip’ or ‘.tar’ file, or as a Git archive, it is best to run it directly from the distribution directory. You need to add the ‘lisp/’ subdirectories to the Emacs load path. To do this, add the following line to your Emacs init file:
💡Orgをウェブから'.zip' や '.tar' ファイル、またはGitアーカイブとしてダウンロードした場合、配布ディレクトリから直接実行するのが一番良いでしょう。Emacsのロードパスに'lisp/'サブディ レクトリを追加する必要があります。これを行うには、Emacsのinitファイルに次の行を追加してください。

(add-to-list 'load-path "~/path/to/orgdir/lisp")

If you have been using git or a tar ball to get Org, you need to run the following command to generate autoload information.
💡もし、Orgをgitやtar ballで取得している場合は、以下のコマンドを実行して、オートロード情報を生成する必要があります。

make autoloads

Activation💡アクティベーション

Add the following lines to your Emacs init file to define global keys for three commands that are useful in any Emacs buffer, not just Org buffers. Please choose suitable keys yourself. 

(global-set-key (kbd "C-c l") #'org-store-link)
(global-set-key (kbd "C-c a") #'org-agenda)
(global-set-key (kbd "C-c c") #'org-capture)

Files with extension ‘.org’ will be put into Org mode automatically.
💡以下の行をあなたのEmacs initファイルに追加して、Orgバッファだけでなく、あらゆるEmacsバッファで有用な3つのコマンドのグローバルキーを定義してください。適切なキーは自分で選ん でください。

Feedback💡フィードバック

If you find problems with Org, or if you have questions, remarks, or ideas about it, please mail to the Org mailing list emacs-orgmode@gnu.org. For information on how to submit bug reports, see the main manual.
💡もしOrgの問題点を見つけたり、質問や指摘、アイディアがあれば、Orgメーリングリスト emacs-orgmode@gnu.org にメールを送ってください。バグレポートの提出方法については、メインマニュアルを参照してください。
